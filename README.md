# hrInRails

Multi-tenant hr application for tracking and displaying PTO.  (Used for learning new Rails concepts and gems)

### Prerequisites

```
Ruby v2.6.5
Rails v6.0.0
 
```

### Installing required Ruby version with rbenv

```
brew install rbenv
rbenv init
rbenv install 2.6.5

cd hrinRails
rbenv local 2.6.5

```

### Setup

```
bundel install
yarn install
rails db:seed
```

### Run

```
rails s
```
